# 哔哩哔哩黑名单同步工具

用于 bilibili 多个帐号之间的黑名单同步

[blocklist.json](blocklist.json) 收集了包括且不限于：流量营销、故意引战、聪明发言、心理年龄幼稚、恶意转载等帐号

## 使用方法

1. 编辑 `config.json`，根据帐号填写配置
2. `bili-blocklist-sync pull` 拉取所有帐号的黑名单
3. `bili-blocklist-sync push` 将黑名单同步到所有帐号
